
# 🚀 JobPulse AI

### AI-Powered Career Intelligence Platform

JobPulse AI is a full-stack Data Engineering and Analytics project designed to transform large-scale job market data into actionable career intelligence.

Built using **1.3M+ LinkedIn Job Postings** and **49K+ Stack Overflow Developer Survey Responses**, the platform provides insights into hiring trends, salary benchmarks, career roadmaps, and emerging AI industry trends.

# 🚀 Installation

Clone repository:

```bash
git clone https://github.com/SarathChandra26/JobPulse-AI.git
```

Navigate:

```bash
cd JobPulse-AI
```

Create environment:

```bash
python -m venv venv
```

Activate:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run local server:

```bash
cd frontend
python -m http.server 8000
```

Open:

```text
http://localhost:8000
```
